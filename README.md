# 6-DOF-Robotic-Arm

This project uses 3 MPU6050 acceleromete, 1 GY-271 Compass sensor and a linear hall effect sensor to operate 6 servo motor.
An I2C multiplexer is used to integrate all the sensor and a PWM driver is used to operate the servo motors.
The brain of this project that does all the calculations is the Arduino Uno.
For power we have used a LiPo battery through a high current buck converter.
